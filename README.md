# Bond Cash Flow Model (Excel/VBA)

## Overview
This repository contains a group university coursework project involving the development of an Excel and VBA model for analysing index-linked bond cash flows and estimating investment yield.

## Project Objective
The aim of the model was to project bond cash flows and evaluate the attractiveness of an index-linked bond investment under different assumptions. The model incorporated inflation effects, tax effects, payment frequency, redemption structure, and purchase conditions.

## What the Model Does
The program allows a user to enter bond parameters through an interactive UserForm and then:
- generates a cash flow table for the bond
- applies inflation-linked adjustments
- applies relevant tax effects
- estimates the annual effective yield using NPV and interpolation methods

## Key Features
- VBA-based cash flow modelling for index-linked bonds
- Interactive UserForm for inputs and execution
- Input validation and VBA-based error handling
- Yield estimation using linear interpolation
- Sensitivity to assumptions such as inflation and purchase price
